# Tarefa: 016 - Tela da Taverna (Tavern)

**Prioridade:** Média
**Tipo:** Feature

## Objetivo
Criar a interface da Taverna, que servirá como o local principal para o jogador encontrar e aceitar missões (quests).

## Requisitos Funcionais

1.  **Layout da Taverna:**
    *   Desenvolver a UI da Taverna, que pode incluir um NPC (o "Taverneiro") e um quadro de missões.
    *   O ambiente deve ser visualmente distinto e convidativo.

2.  **Lista de Missões (Placeholder):**
    *   Implementar uma UI que exiba uma lista de missões disponíveis.
    *   Para esta tarefa inicial, a lista pode ser um mock (dados falsos), já que o sistema de missões completo ainda não foi projetado.
    *   Cada item na lista deve mostrar, no mínimo, o título da missão e o nível recomendado.

3.  **Interação:**
    *   Permitir que o jogador clique em uma missão para ver uma breve descrição.
    *   Um botão "Aceitar" pode ser incluído, mas sua funcionalidade completa dependerá do futuro sistema de quests.

## Nota
O foco desta tarefa é criar o **ambiente e a UI** da Taverna. A lógica complexa de um sistema de missões (rastreamento de progresso, condições, recompensas) será definida em uma tarefa futura.

## Critérios de Aceitação
*   O jogador pode acessar a Taverna a partir da Praça Central.
*   A tela exibe um layout temático de taverna com uma lista de missões de exemplo.
*   É possível clicar em uma missão para ver seus detalhes (mock).
