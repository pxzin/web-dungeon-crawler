# Tarefa: 018 - Definição de Dados dos Monstros (Cripta)

**Prioridade:** Média
**Tipo:** Conteúdo

## Objetivo
Criar as definições de dados para os monstros mortos-vivos que habitarão a segunda masmorra, "A Cripta dos Inquietos".

## Requisitos Técnicos

1.  **Adicionar ao Arquivo de Conteúdo:**
    *   As novas definições de monstros devem ser adicionadas ao arquivo `src/lib/content/monsters.ts`.

2.  **Criação dos Monstros Mortos-Vivos:**
    *   Criar as definições para os seguintes monstros, com estatísticas e habilidades apropriadas para a faixa de nível 5-10:
        *   **`skeleton_warrior`**: Um guerreiro esqueleto básico, resistente a perfurações, mas talvez vulnerável a dano de esmagamento.
        *   **`wraith`**: Um fantasma que pode ter ataques que drenam recursos (mana/energia) ou aplicam status negativos (ex: lentidão).
        *   **`zombie_hulk`**: Um zumbi lento, com muita vida e um ataque poderoso, mas impreciso.
        *   **`acolyte_necromancer` (Chefe):** O chefe da masmorra. Um necromante que pode invocar esqueletos ou usar magias sombrias.

## Critérios de Aceitação
*   As definições para os 4 novos tipos de monstros mortos-vivos são adicionadas ao arquivo de conteúdo.
*   Cada monstro tem um conjunto de estatísticas balanceado para a faixa de nível 5-10.
*   O chefe "Acolyte Necromancer" é notavelmente mais desafiador que os outros monstros.
