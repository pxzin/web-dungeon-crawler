# Checklist de Assets Gráficos

Este documento rastreia todos os assets gráficos necessários para o projeto, com base no GDD e nas tarefas planejadas.

## Legenda de Status
-   ✅ **Existente / Concluído**
-   ⏳ **Necessário / Pendente**

---

### 1. Interface de Usuário (UI) e Ambientes

-   **✅ Design System Base:** Botões, cards, modais, etc. (Implementado via código)
-   **⏳ Background da Praça Central:** Ilustração principal do hub.
-   **⏳ Background do Ferreiro:** Ilustração de interior de uma forja.
-   **⏳ Background da Taverna:** Ilustração de interior de uma taverna.
-   **⏳ Background do Mercado:** Ilustração de uma loja ou barraca de mercado.
-   **⏳ Background de Combate:** Cenário(s) de fundo para a tela de combate.
-   **⏳ Tileset de Caverna:** Para a "Caverna dos Goblins".
-   **⏳ Tileset de Cripta:** Para a "Cripta dos Inquietos".
-   **⏳ Outros Tilesets:** (Floresta, Templo, etc., conforme o roteiro do GDD).

### 2. Ícones

-   **✅ Ícones de Itens/Habilidades:** Spritesheet `rpg-icons.png`.
